# 🤖 Next-Generation AI Chatbot

A modern AI-powered chatbot built with Google's Gemini 2.5 Flash SDK and Streamlit, featuring real-time response streaming, dynamic AI personas, session memory, and a premium glassmorphism user interface.

---

# 🚀 Features

## ✅ Real-Time AI Streaming

* Live response rendering using `st.write_stream`
* Smooth and interactive chatting experience

## ✅ Dynamic AI Personas

Switch chatbot behavior instantly with multiple modes:

* 💻 Coding Expert
* 📈 Business Consultant
* 🎨 Creative AI
* 🧠 Strategic Advisor

## ✅ Active Session Memory

* Maintains conversation history
* Provides accurate contextual responses

## ✅ Premium Glassmorphism UI

* Modern custom CSS design
* Blurred sidebar effects
* Gradient backgrounds
* Styled input components

# 📦 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 3️⃣ Add Gemini API Key

Open your Python file and add your API key:

```python
API_KEY = "YOUR_API_KEY"
```

---

# ▶️ Run Project

```bash
streamlit run app.py
```
